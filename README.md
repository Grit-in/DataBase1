## Description

This project was created for the **Databases 1 course** to demonstrate the use of MongoDB and dynamic server-rendered web applications. It functions as a **phone catalog system**, displaying detailed pages for each phone stored in the database.

The application uses **Mongoose** for MongoDB interactions and leverages various **npm packages** for functionality and styling, including **Bootstrap** for responsive UI.

> **Note:** The project requires an active server and MongoDB connection to function properly.

### Features

- **Dynamic Webpages** for each phone entry.
- **MongoDB Integration** using Mongoose.
- **Clean UI** with Bootstrap and custom styles.
- **Modular Codebase** using Express.js and EJS for rendering.

## Technologies Used

- **Backend:**
  - Node.js
  - Express.js
  - MongoDB
  - Mongoose
- **Frontend:**
  - HTML5
  - CSS3
  - JavaScript
  - Bootstrap
- **Templating:**
  - EJS
